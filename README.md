# Simulador de Portafolio

Este proyecto es un simulador que combina **inversión en acciones** y **renta fija (CDTs/bonos)** para mostrar cómo evoluciona un portafolio financiero en el tiempo.

## Funcionalidades
- **Acciones**:
  - Compra y venta de acciones.
  - Registro de movimientos.
  - Gráfica de evolución de precios históricos (ejemplo: Apple).

- **Renta fija**:
  - Simulación de intereses diarios con capitalización compuesta.
  - Gráfica de crecimiento del dinero invertido.

- **Portafolio total**:
  - Dinero disponible + acciones + renta fija.
  - Cálculo del valor total del portafolio.
